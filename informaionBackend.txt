Pasos para el backend:

instalar sequilize:
npm install sequelize sqlite3
para comprobar su correcta instalación:
npm init -y

instalar express e incluir librería node:
npm install express sequelize pg pg-hstore dotenv
npm install -D typescript ts-node-dev @types/node @types/express

instalar typescript y tsx
npx tsc --init
npm install -D tsx

inicializar el servidor: 
npm run dev

Estructura del backend:
/backend
	/src
		database.ts
		index.ts
		/models 	*Aquí está el modelo de las tablas que se subirá a postgresql*
			Caja.ts
			Disponibilidad.ts
			Empresa.ts
			Reserva.ts
			.....etc
		/routes		*Aqui estan todos los endpoints del sistema*
			role.routes.ts
			usuario.routes.ts
			....etc
	.env
	packager-lock.json
	package.json
	tsconfig.json
