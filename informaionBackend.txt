Pasos para el backend:

instalar sequilize:
npm install sequelize sqlite3
para comprobar su correcta instalación:
npm init -y

instalar express e incluir librería node:
npm install express sequelize pg pg-hstore dotenv
npm install -D typescript ts-node-dev @types/node @types/express

instalar typescript y tsx
npx tsc --init
npm install -D tsx

inicializar el servidor: 
npm run dev

Estructura del backend:
/backend
	/src
		database.ts
		index.ts
		/models 	*Aquí está el modelo de las tablas que se subirá a postgresql*
			Caja.ts
			Disponibilidad.ts
			Empresa.ts
			Reserva.ts
			.....etc
		/routes		*Aqui estan todos los endpoints del sistema*
			role.routes.ts
			usuario.routes.ts
			....etc
	.env
	packager-lock.json
	package.json
	tsconfig.json

instalar  multer para guardar el logos y qr de empresa

instalar brcryp para hashear/incriptar el password 
npm install bcrypt
npm install -D @types/bcrypt

cors para la conectividad de puertos
npm install cors
npm install -D @types/cors

para agregar decoradores en las tabals sequelize.
npm install sequelize sequelize-typescript mysql2
npm install -D @types/node

instalación JWT
npm install jsonwebtoken

instalación cloudinary
